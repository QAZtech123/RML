# Documentation Index

This directory is the public-facing technical documentation set for the RML engine.

## Documents
- `architecture.md`: component architecture, training loop, and telemetry flow
- `mathematical_reasoning.md`: objective geometry, guard conditions, and reward equations
- `new_contributions.md`: concrete features added in the current RML development cycle
- `results.md`: current empirical outcomes, non-stationarity notes, and claim boundaries
- `publishing.md`: clean GitHub checklist and release packaging workflow

## Scope
- Explain what the system does today.
- Separate confirmed results from hypotheses.
- Keep claims aligned with evidence in `rml_lab_log.md`.
